# Codex Switch v2.1.1

Small UI release on top of the v2.1 account-persistence hardening.

## Changes

- Shows the app version in the popover header.
- Keeps the v2.1 account-persistence hardening: single-instance lock, identity-safe account sync, atomic writes, login polling, and regression tests.

## Install

Download `Codex-Switch-v2.1.1.dmg`, open it, and copy `Codex Switch.app` to Applications.

This release is not signed or notarized with an Apple Developer account. macOS may require manually allowing the app in System Settings the first time it is opened.
