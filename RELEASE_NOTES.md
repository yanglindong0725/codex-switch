# Codex Switch v1.0.1

Stability and privacy-focused maintenance release.

## Changes

- Removed the unredacted README screenshot from the published Git history.
- Fixed the disabled auto-refresh option so it does not create a zero-second timer.
- Made account switching safer by staging replacement files before updating `auth.json`.
- Added visible errors for failed settings saves and login item registration.

## Install

Download `Codex-Switch-v1.0.1.dmg`, open it, and copy `Codex Switch.app` to Applications.

This release is not signed or notarized with an Apple Developer account. macOS may require manually allowing the app in System Settings the first time it is opened.
