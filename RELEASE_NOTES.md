# Codex Switch v1.0.0

First independent open-source release of Codex Switch.

## Changes

- Renamed the public app identity to Codex Switch.
- Updated the macOS app bundle identifier to `com.henry.codex-switch`.
- Added a release script that builds `Codex-Switch-v1.0.0.dmg`.
- Preserved MIT License attribution for the original `jieguangzhou/CodexSwitcher` project.

## Install

Download `Codex-Switch-v1.0.0.dmg`, open it, and copy `Codex Switch.app` to Applications.

This release is not signed or notarized with an Apple Developer account. macOS may require manually allowing the app in System Settings the first time it is opened.
